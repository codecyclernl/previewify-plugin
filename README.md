# Previewify October CMS plugin

Your competitor isn't designing those images by hand. They're generating them via platforms like this one. They spend less time and have a bigger conversion rate.

More information about Previewify is available on their website [previewify.app](https://previewify.app).

This plugin integrates with October CMS and most popular plugins. It's even easier to install Previewify on your October CMS website.

## Installation

Use the package manager [composer](https://getcomposer.org/) to install the Previewify plugin.

```bash
composer require codecycler/previewify-plugin
```
or
``` bash
php artisan plugin:install Codecycler.Previewify
```

You can now use the `Previewify Image` component on your layouts. Drag and drop the component inside the `head` tags of your template.

## Usage
Click on the `Previewify Image` component in your layout to adjust some settings.

`Title` -> leave empty to display the page meta title

`Subtitle` -> leave empty to display the page meta description
